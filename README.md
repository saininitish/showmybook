# showmybook

A Next.js movie and event ticket booking app (ReelBook).

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech

- Next.js 14, React, TypeScript
- Supabase (auth, DB)
- Tailwind CSS
- EmailJS (booking & contact emails)
- Groq (AI chat)

## Deploy

You can deploy on [Vercel](https://vercel.com) or any Node host. Set environment variables (see `.env.local.example`).
